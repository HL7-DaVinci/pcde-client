PCDE Client

This server provides a client for interacting with the Da Vinci Payer Coverage Decision Exchange Reference Implementation.

To use a different part of the client click on one of links on the side menu.

FHIR Endpoint

All requests are made to the endpoint: https://davinci-pcde-ri.logicahealth.org/fhir/PCDE

Note that this is not a traditional FHIR endpoint but is created as a custom endpoint to respond with the communication when a communication request is recieved

To send a sample communication request that will receive a valid communication with the PCDE Bundle, the patient ID needs to be 14 and the Organization IDs should be 16 and 17
